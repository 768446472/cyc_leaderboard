📖 如何获取赛段 ID？

从 Strava 赛段页面 URL 中提取数字 ID，有3种方法：

1
网页版 Strava
在 strava.com 打开任意赛段页面，浏览器地址栏的 URL 中包含数字：
https://www.strava.com/segments/33862358
→ 赛段 ID 就是 33862358
2
Strava 手机 App
进入赛段详情页 → 点击右上角 分享 → 复制链接，链接格式同上，提取其中的数字。
3
俱乐部 / 活动页面
赛段名称旁边通常有链接，右键 复制链接地址，格式为：
https://www.strava.com/segments/3591075?filter=overall
→ 提取 3591075
