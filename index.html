📋 按赛段 ID 打开

输入 Strava 赛段 ID，直接跳转到对应分析报告。

📖 如何获取赛段 ID？

从 Strava 赛段页面 URL 中提取数字 ID：

1
网页版 Strava
打开任意赛段页面，地址栏 URL：
https://www.strava.com/segments/33862358
→ 赛段 ID = 33862358
2
Strava 手机 App
赛段详情页 → 右上角 分享 → 复制链接 → 同格式提取数字。
3
俱乐部 / 活动页面
赛段名称旁右键 复制链接地址：
https://www.strava.com/segments/3591075?filter=overall
→ 赛段 ID = 3591075

🧪 使用工具生成报告

通过 Python 脚本抓取 Strava 排行榜数据并生成 HTML 分析报告。

前置条件

1. Python 3.8+ 及依赖：pip install stravalib requests beautifulsoup4

2. 从浏览器导出 Strava Cookie 保存为 cookie.txt（Netscape 格式）

命令行

python strava_segment_analyzer.py 33862358 demo --cookie-file cookie.txt

可选：--max-pages N 控制抓取页数（每页约100条）。
